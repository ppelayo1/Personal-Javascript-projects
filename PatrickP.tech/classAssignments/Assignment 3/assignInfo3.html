Assignment 3

Assignment 3 is a continuation of Assignment 2. The instructor provides the layout of the site, I implement the text, images, and javascript. The web page is converted to utilize event handling, and the form tag.

Part 1

Part 1 involves using an inline onclick event on the button to read the value in the form field for the name. It then outputs a message thanking the user for their order. No other actions occur in this part.

Part 2

Part 2 continues from part 1.
The user fills out all fields, all fields utilize input masking to prevent inaccurate data from entry. The name field allows only characters, single spaces, single ., tabing, and backspacing. The item amount fields only allows numerical values, tabs, and backspaces.
When the button is clicked it executes a function that thanks the user for their order, and presents the number items ordered in that message.

Part 3

Part 3 continues from part 2.
The main differance is that the onclick is handled by an event listener for the button. The button now outputs a more detailed receipt to the user.
