Assignment 2

Assignment purpose is to provide experience with conditional operators and logic.

Part 1

An alert for a first name, and then for the person's GPA. As output the letter Grade is provided.
I performed input validation by disallowing non-letter input and truncating any last names.

Part 2

Layout provided by instructor, I only needed to provide the images, and text to the page.
This page is to resemble a store page in the purchase of two differant items. Alerts are used to gather the request information, and I implement input validation of the data. As output the number of items ordered is presented to the user.

Part 3

Same as Part 2 ,but outputs the total cost of each item, and the grand total of the purchase.

Part 4

Same as Part3, but I include taxes on the order.

Part 5

Same as Part 4, but I use a function to handle the output of the sale receipt.
